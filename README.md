# TG ADMIN PANEL
Сервис для общения администратора с 
пользователями через телеграм.

Подготовка:
```cmd
1. Создать dev.env файл в папке env
2. python -m venv venv
3. source venv/bin/activate
4. pip install -r req.txt
```

Запуск проекта (из корня):
```cmd
python main.py
```
Запуск тестов:
```cmd
python test.py
```
